# Customer Behavior Analysis

## Project Overview

This project was completed as part of the Alfido Tech Data Analyst Internship. The objective of this project is to analyze customer purchasing behavior using Python and Exploratory Data Analysis (EDA). RFM (Recency, Frequency, Monetary) Analysis is also performed to segment customers and generate business insights.

---

## Objectives

- Load and clean the customer dataset.
- Perform Exploratory Data Analysis (EDA).
- Visualize customer purchasing patterns.
- Perform RFM Analysis.
- Segment customers based on purchasing behavior.
- Generate business insights and recommendations.

---

## Dataset

- Dataset Name: Customer Behavior Analysis
- Source: Kaggle
- Dataset Link: https://www.kaggle.com/datasets/bhanupratapbiswas/customer-behavior-analysis

## Key Findings

- Customer purchasing behavior differs across product categories.
- Revenue trends vary across different months.
- High-value customers contribute significantly to total revenue.
- Customer segmentation helps identify loyal and at-risk customers.
- Payment methods and purchasing patterns provide valuable business insights.

---

## Business Recommendations

- Reward high-value customers through loyalty programs.
- Re-engage at-risk customers using personalized marketing.
- Promote high-performing product categories.
- Reduce product return rates by improving product quality.
- Use customer purchasing behavior to improve marketing strategies.
